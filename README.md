## Features
### 1. 使用者可以瀏覽所有餐廳的資訊：餐廳名稱、餐廳圖片、料理類別、評價。
### 2. 使用者可以觀看特定餐廳詳細介紹，得知餐廳地址、電話號碼、餐廳簡介。
### 3. 使用者可以使用搜尋功能根據自訂的關鍵字尋找特定餐廳。

## How to use
### 1. 在 Terminal 上啟動專案。
`npm run dev`

`nodemon app.js`
### 2. 載入seed
`npm run seed`
